### ⚙️ 対象システムの仕様・概要
# ターボチャージャーシステムの仕様・概要

ターボチャージャー（排気タービン過給機）は、本来捨てられるはずの**エンジンの排気ガスが持つエネルギーを利用してコンプレッサーを回し、エンジンに大量の空気を強制的に送り込む（過給する）システム**です。

同じ排気量の自然吸気（NA）エンジンに比べて、より多くの燃料を効率よく燃焼させられるため、**大幅な出力・トルクの向上**や、排気量を小さくして燃費を稼ぐ**ダウンサイジング**を可能にします。

---

## 1. システムを構成するコア部品

ターボチャージャーは主に以下のコンポーネントで1つのシステムとして機能します。

* **タービンホイール（排気側）**
  * 排気マニホールドから流れてくる排気ガスの勢いで、毎分10万〜20万回転以上の超高速で回転します。
* **コンプレッサーホイール（吸気側）**
  * タービンと一本の「センターシャフト（軸）」で直結しており、連動して超高速回転することでエアクリーナーからの空気を圧縮します。
* **ベアリング（軸受け）ハウジング**
  * センターシャフトを支える部分です。超高速回転・超高温に耐えるため、エンジンオイルを循環させて常に潤滑と冷却を行っています。
* **インタークーラー**
  * 圧縮されて高温（100℃〜150℃以上）になった吸入空気を冷却する装置です。空気を冷やすことで空気密度が上がり、より多くの酸素をシリンダーに送り込めるようになります。

---

## 2. ターボチャージャーの動作プロセス

システムが空気を過給する流れは以下の4ステップです。

1. **エネルギー回収**
   * エンジンが燃焼した後の排気ガスが、エキゾーストマニホールドを通ってタービンハウジングへ流れ込みます。
2. **過給の開始**
   * 排気ガスの圧力でタービンホイールが回り、その回転力がシャフトを伝って吸気側のコンプレッサーホイールを回します。
3. **空気の圧縮**
   * コンプレッサーが外気を強力に吸い込み、圧力を高めて（過給圧・ブースト圧）送り出します。
4. **燃焼・出力向上**
   * 圧縮された空気はインタークーラーで冷やされた後、エンジンのシリンダー内へ押し込まれ、NAエンジン以上の爆発的なパワーを生み出します。

---

## 3. 主要な制御・安全装置（補機類）

過給圧（ブースト圧）が上がりすぎるとエンジンやタービン本体が破損するため、以下の制御システムが組み込まれています。

* **ウェストゲートバルブ**
  * 排気側のバイパス弁です。設定したブースト圧に達すると弁が開き、排気ガスの一部をタービンを迂回させて逃がし、過給圧を一定に保ちます。
* **ブローオフバルブ（バイパスバルブ）**
  * アクセルを戻した際、行き場を失った高圧の吸入空気を吸気配管内から逃がす弁です。コンプレッサーに逆流して羽を痛めるのを防ぎます。

---

## 4. 主な仕様・バリエーション

エンジンの特性や用途に合わせて様々な仕様が存在します。

| 仕様・タイプ | 概要と特徴 |
| :--- | :--- |
| **シングルターボ** | 1つのエンジンに1つのターボを組み合わせる最も標準的な構成。 |
| **ツインターボ** | 2つのターボを使用。低回転用と高回転用で分ける「シーケンシャル方式」や、左右のバンクに配置する「並列方式」があります。 |
| **ツインスクロールターボ** | 排気ガスが通る通路（スクロール）を2つに分け、排気干渉を抑えて低回転からのレスポンスを高めた仕様。 |
| **VGT（可変ジオメトリターボ）** | タービン内の可変フラップを動かし、排気ガスの流速をコントロールする仕様。主にディーゼル車に多く採用されます。 |

---

## 5. メリットとデメリット

### メリット
* **高出力・大トルク**: 排気量以上のパワーを発揮でき、加速性能が大幅に向上します。
* **ダウンサイジング**: 排気量を小さくできるため、自動車の税金を抑えつつ、エンジン自体の軽量化や摩擦損失を減らして環境性能を高められます。

### デメリット
* **ターボラグ**: アクセルを踏んでから排気ガスが溜まり、過給が始まるまでに一瞬の時間差（遅れ）が生じます。
* **熱とオイル管理への負荷**: 超高温・超高速で回転するため、定期的なエンジンオイル交換を怠ると軸受けが焼き付く原因（タービンブロー）になります。
