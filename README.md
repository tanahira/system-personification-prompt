# システム構成を擬人化するプロンプト
複雑なシステムを「人」や「チーム」に例えて理解しやすくするAIプロンプト。

複雑なシステム構成やソフトウェアの役割分担を、キャラクターやチームに置き換えて説明させるためのプロンプトです。

システム設計書や仕様書をそのまま入力するだけで、各機能の役割や連携を直感的に理解できるようになります。

---

## なぜ作ったのか

システム構成図や仕様書は、開発者には理解しやすくても、初学者や非技術者には全体像が見えにくいことがあります。

そこで、

* 小学生でも理解できる
* 新入社員でも理解できる
* 営業職や事務職でも理解できる

ことを目標に、システムの各機能を「人」に例えて説明するプロンプトを考案しました。

---

## プロンプト

```text
以下のシステムについて、登場する機能やデバイスをすべて擬人化して、チームや会社としての役割分担に例えて説明してください。

小学生でもそれぞれの役割や全体の連携の流れがイメージできるように、わかりやすくお願いします。

# [ここにシステムの概要や仕様書を貼り付ける]
```

---

## 使用例

### 入力

```text
ESP32-CAMが画像を撮影し、
Flaskサーバへ送信する。

サーバは画像を解析し、
SQLiteへ結果を保存する。
```

### 出力イメージ

* ESP32-CAM：現場カメラマン
* Flaskサーバ：受付係
* 画像解析AI：検査担当
* SQLite：記録保管担当

みんなで協力して仕事を進める会社として説明される。

## AIイラスト生成との連携

本プロンプトは文章による説明だけでなく、AI画像生成との組み合わせも想定しています。

擬人化された役割をもとに画像生成AIへ入力することで、システム全体をチームや組織として視覚的に表現できます。

### 例

システム構成：

* ESP32-CAM：現場カメラマン
* Flaskサーバ：受付係
* 画像解析AI：品質検査員
* SQLite：記録保管担当

↓

AI画像生成

「現場カメラマン、受付係、品質検査員、記録保管担当が協力して働く会社のイラスト」

システム構成図が苦手な人でも、役割や連携を直感的に理解しやすくなります。
